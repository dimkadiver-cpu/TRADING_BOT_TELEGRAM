# listener_sidecar — RIMOSSO (PRD 2.c)

`RuntimeV2ListenerSidecar` è stato eliminato in PRD 2.c. La variabile d'ambiente `USE_RUNTIME_V2` non è più necessaria.

Per avviare il bot con la pipeline runtime_v2:

```bash
python main.py
```

Il runtime_v2 è sempre attivo — non richiede flag. Vedi `docs/runtime_v2/overview.md`.
