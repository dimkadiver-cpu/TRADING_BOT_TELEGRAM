# Hummingbot Demo Connector Patch

This directory contains a patched `bybit_perpetual_constants.py` that adds
`bybit_perpetual_demo` pointing to `https://api-demo.bybit.com`.

## Usage

Mounted read-only into the Hummingbot demo container via docker-compose.demo.yml.

## Before first use

Run SETUP.md instructions to verify the mount path against the pinned image version.
